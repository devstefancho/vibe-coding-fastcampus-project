## Playright MCP 설정
- Playwright MCP 설치하기 -> claude code 에게 검색하라고 하고 설치해달라고 해보자 ("playwright mcp 설정해줘 https://github.com/microsoft/playwright-mcp 참고해서")
- Claude Code 공식문서에서 MCP 설치방법 한번 읽어보기 (중요한건 아닙니다. 근데 명령어를 좀 더 잘알고 싶으시면 한번쯤 보면 좋습니다. 두번세번 보면 익숙해지기 시작합니다.)
- /mcp 명령어로 현재 사용가능한 mcp 서버리스트 확인하기
- claude mcp add 명령어 -> 이거는 솔직히 외워도 까먹습니다. 그냥 claude code 시켜서 설치하세요. 문서만 주면 됩니다.
- MCP 가 정상적으로 설치되지 않은 경우 어떻게 나오는지? 그리고 정상적 설치가 안되었을때는 claude code에게 mcp 연결안되는 이유가 뭔지 확인해줘 라고 해봅시다. (직접 해결하기 보다는 항상 claude code에게 try 해보는겁니다.)


## Playwright MCP 활용
Playwright MCP 사용시켜보기
- Claude Code에게 Playwright MCP가 어떤걸 할 수 있는지 물어봅시다. 이게 제일 중요합니다. 활용방법을 구글링하기보다는 ai한테 직접 물어보는겁니다. 단순한 질문도 상관없습니다.
- 내가 브라우저에서 할법한 동작들을 다 시켜봅니다. ex) 브라우저 열기, 특정 주소로 이동하기, 특정 버튼 클릭하기, 스크린샷 찍기
- 상황 이해가 필요한 동작에 대해서 얘기해도 됩니다. ex) 구글에서 바이브 코딩 검색해줘



내가 만든 페이지를 열어보게 만들어봅시다.
- 샘플 코드를 하나 만들어봅시다. 샘플코드는 실패하지 않는 예시로 만들어야하므로 왠만하면 실패하지 않는 날씨앱으로 만들겠습니다. ("Vite, Typescript로 간단한 날씨앱 하나 만들어줘")
- 단, API 연동은 하지 않고, 가짜 데이터로 해달라고 하겠습니다. ("가짜 데이터로 만들어줘, 그리고 완성하면 서버실행해줘")
- 잘 생성되었으면 본인이 직접 열어봅니다.
